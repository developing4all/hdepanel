# HDE Panel
Qt5 based panel based on QtPanel 


# Features
Very light
Beautiful look.
Multiple panels
Multple screen support
Xdg compatible
Per panel configuration dialog
Support for plugins

# Plugins
Applications menu.
Taskbar with drag-and-drop rearrange.
Notification area (tray).
Clock applet.

GPL 2.0


# Build 
qmake
make

# Run
export LD_LIBRARY_PATH=./:$LD_LIBRARY_PATH
./hdepanel
